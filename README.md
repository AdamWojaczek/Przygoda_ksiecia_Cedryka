# PRZYGODA KSIĘCIA CEDRYKA

Gra "Przygoda księcia Cedryka" to edukcyjna adaptacja mini przygody (pod tym samym tytułem) pochodzącej z gry fabularnej (RPG) napisanej dla polskiego systemu "KRYSZTAŁY CZASU" autorstwa Artura Szyndlera, która została opublikowana w czsopiśmie "Magia i Miecz" nr 1 w roku 1993.

Niniejsza (bardzo prosta, wręcz trywialna) gra paragrafowa/narracyjna, powstała w celach edukacyjnych, jako wprawka (i zabawa) w programowaniu w języku Python.
Jest to obiektowa wersja program, który napisałem w czasach licelnych (styczeń 1996 r.) w języku QBASIC (również w ramach edukacji).
