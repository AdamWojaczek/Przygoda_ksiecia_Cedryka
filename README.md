# PRZYGODA KSIĘCIA CEDRYKA

Gra "Przygoda księcia Cedryka" to adaptacja mini przygody, pochodzącej z gry fabularnej (RPG) napisanej dla polskiego systemu "KRYSZTAŁY CZASU" autorstwa Artura Szyndlera, która została opublikowana w czasopiśmie "Magia i Miecz" nr 1 w roku 1993.

Niniejsza krótka i bardzo prosta gra paragrafowa/narracyjna, powstała w celach edukacyjnych, jako wprawka (i zabawa) w programowaniu w języku Python.
Jest to obiektowa wersja program, który napisałem w czasach licelnych (w styczniu 1996 r.) podczas nauki jezyka QBASIC.
